# Skills

Agent skills maintained by hi-bjHuang.

## Available skills

### vue-component-split

Guidance for splitting large Vue 3 single-file components into focused child components while preserving behavior.

Install with:

```bash
npx skills add hi-bjHuang/skills --skill vue-component-split
```

Install all skills from this repository:

```bash
npx skills add hi-bjHuang/skills
```

After this repository is installed through the `skills` CLI, skills.sh can discover it through anonymous install telemetry and create/update the page at:

https://skills.sh/hi-bjHuang/skills
